# Workflow

## User Journey

1. User opens SuccessPilot AI.
2. User enters customer meeting notes or uploads meeting transcripts.
3. AI analyzes the input.
4. AI generates:
   - Meeting Summary
   - Action Items
   - Customer Health Insights
   - Follow-up Email
5. User reviews and copies the generated output.
6. User shares the output with customers or updates their CRM.
