# SuccessPilot AI

> AI-powered Customer Success Assistant built using no-code Generative AI tools.

## 📖 Overview

SuccessPilot AI is a portfolio project that demonstrates how Generative AI can help Customer Success teams automate repetitive tasks such as meeting summaries, customer health analysis, follow-up emails, and account insights.

The project is designed using no-code AI tools and focuses on improving productivity, customer engagement, and decision-making.

---

## 🎯 Objectives

- Automate meeting summaries
- Generate follow-up emails
- Analyze customer health
- Identify customer risks
- Recommend next best actions
